# COMP311 – Linux Operating System Labs 🐧💻

This repository contains lab solutions, projects, and reference manuals for the **Linux OS course (COMP311)** at Birzeit University. It includes hands-on exercises related to the Linux command line, scripting, file permissions, process control, and more.

---

## 📁 Folders Overview

- **project1 / project2 / project3** – Lab projects involving file handling, command execution, scripting, etc.
- **حل كل لابات عملي** – Complete solutions for all Linux lab sessions (in Arabic).
- **سلايدات دكتور مراد** – Instructor slides for Linux theory and practice.
